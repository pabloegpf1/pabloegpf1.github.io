Communications-Reporting Tool

Verano 2019

Communications-Reporting Tool es una herramienta web que desarrollé durante mis prácticas en IMDEA Networks que permite analizar el impacto producido por investigaciones científicas en los medios de comunicación. El principal motivo para desarrollar esta aplicación es facilitar el seguimiento de toda la información publicada en los medios de forma sencilla y eficaz, de forma que se pueda extraer datos y estadísticas en tiempo real y diferentes formatos.

CosmoQuiz

Mayo 2019

CosmoQuiz es una app para iOS que usa la API de la NASA para generar imágenes astronómicas diarias (APOD por sus siglas en inglés). CosmoQuiz permite a los usuarios resolver cuestionarios relacionados con estas imágenes mientras aprende sobre nuestro universo. Programado en Swift, utiliza Firebase como base de datos / sistema de autenticación y la plataforma Nasa Open APIs para el contenido.
